About this project
I worked for a non-profit news organization that gets thousands of views each month. As a web developer, I was in charge of three main things - CSS Styling, UI/UX design, and site speed design. For CSS styling, I had to style website elements to be visually appealing and display correctly on multiple devices including smartphones. For UI/UX design, I had to make sure that my CSS styles made the UI easy for users to navigate and use the website. Finally for site speed optimization, I used a combination of techniques such as using caching plugins like WP-Rocket with file minimization to optimize loading times of the website.
Technologies Used
  • Wordpress
  • WP Rocket (Caching and File minimization)
  • CSS3 - (Media Queries, CSS Grids)
  • PenNews
